# kraken
GraphQL server boilerplate

![](https://api.travis-ci.org/kodjevlar/kraken.svg?branch=master)

## Requirements
- docker
- docker-compose

## Development setup
`npm install`<br/>
`docker-compose up`

### Running tests
Run tests once:
`npm test` or `npm run test`<br/>

Run and watch tests:
`npm run test:watch`
